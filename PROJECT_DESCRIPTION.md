# Juno - Advanced MERN B2B Teams Project Management SaaS

## Project: 
Juno - Advanced MERN B2B Teams Project Management SaaS

## Role: 
Full-stack Developer

## Tech Stack: 
React, TypeScript, Vite, Node.js, Express, MongoDB, Mongoose, Docker, Kubernetes, Nginx, GitHub Actions, GKE, Google Cloud Platform, OAuth (Google), JWT, RESTful APIs, TailwindCSS, React Hook Form, Zod, TanStack Query, shadcn/ui

## Description: 
Built a comprehensive B2B project management SaaS platform with multi-tenant architecture, allowing teams to collaborate on projects, manage tasks, and track progress. Implemented end-to-end functionality including user authentication with Google OAuth, role-based access control, workspace management, project creation, and task tracking. The application features a modern React frontend with responsive UI and a robust Node.js/Express backend with MongoDB database integration.

## Achievements: 
- Architected and implemented complete containerization with Docker for both frontend and backend services
- Deployed and orchestrated the application using Kubernetes with proper deployments, services, and ingress configuration
- Set up automated CI/CD pipeline with GitHub Actions for continuous deployment to Google Kubernetes Engine
- Implemented sophisticated role-based access control system with customizable permissions
- Integrated Google OAuth for seamless user authentication and authorization
- Optimized the application for scalability with proper resource limits and health checks in Kubernetes
- Created comprehensive API endpoints for full CRUD operations across all entities (users, workspaces, projects, tasks)
- Designed and implemented intuitive user interface with responsive design principles
